About driverbot

The driverbot project started in March 2024. It's a school project aimed to create a small-scale Lego-based car. The technologies used are MQTT, ESP8266 microcontroller, motors, servos and cad-modelled parts.

The requirments were as follows, the car should be:

  • Able to drive/steere forward, left, right and backwards.
  • Controllable via a website.
  • At max 15x15x15 centimeters in size.

If you want to take a look at the code for this proejct, head to my GitHub repository HERE.
